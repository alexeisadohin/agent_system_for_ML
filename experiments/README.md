Здесь будет размещаться код экспериментов агентов


Архитектуры агентных систем:
- Один агент (ML инженер)
- Два агента: аналитик данных(задача - предобработать датасет) и ML инженер(задача - опробовать разные модели и получить значение метрики)
- Два агента + руководитель: та же самая схема, только добавляется отдельный агент руководитель, который даёт задания двум другим агентам(при этом руководительн только принимает решения и не имеет каких-либо инструментов кроме как вызов других агентов)
- ...

Также для каждого из агентов можно добавлять:
- Планирование своих действий (в smolagents выставляется планирование раз в N шагов)
- Подсказки общего характера (без привязки к конкретному датасету)



Для каждой архитектуры на каждом датасете собирались следующие метрики:
- Показатель метрики на тестовой выборке
- Время работы агента
- Количество затраченных токенов:
  - Входных
  - Выходных
